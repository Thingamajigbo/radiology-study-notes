01CT 영상 재구성 파이프라인

전체 흐름 (순서 암기 필수)

  • 검출기 신호
  • → 측정 데이터 (measurement)
  • → [전처리] 보정·로그변환
  • → 원 데이터 (raw) ⭐ 전처리 직후
  • → 투영 데이터 (projection/사이노그램)
  • → [컨볼루션] ⭐ 여기서 커널 적용
  • → 필터 된 데이터 (filtered)
  • → [역투영]
  • → 이미지 데이터 (image)
  • → [후처리] MPR/MIP/VR/3D

3단계 구분

  • 전처리: 물리적 보정 (measurement → raw)
  • 재구성: 컨볼루션 + 역투영 (raw → image)
  • 후처리: 3D 재구성 (image → MPR/3D)

02재구성 알고리즘(커널) 매칭

알고리즘 계열 스펙트럼

  • Soft / Smoothing / Soft tissue → 부드러움
  • Standard → 중간
  • High / Lung → 강 sharp
  • Bone → 매우 강 sharp
  • Ultra-high / Edge / Bone Plus → 극강 sharp

부위별 매칭 (시험 최빈출!)

계열부위
Soft / Smoothing / Soft tissue복부, 뇌실질, 종격동
Standard일반 routine, 뇌·복부 일반
High / Lung폐 HRCT
Bone척추, 골반, 두개골, 부비동
Ultra-high / Edge관자뼈(측두골), 내이
🎯 시험 함정 대비
  • 보기에 Soft와 Standard 둘 다 있을 때: 맥락 판단
  • "일반 조직 / routine" → Standard
  • "연부조직 특화 / 저대조도" → Soft
  • 한 줄 원칙: "대조도 크면 sharp, 대조도 작으면 soft"

03Window Width / Level (창폭/창높이)

부위별 표준 설정 (숫자 하나로 암기!)

부위WW/WL특징
뇌실질100 / 30 가까운 것좁고 낮음
복부400 / 40 가까운 것중간
폐1500 / -500 가까운 것넓고 음수
뼈2000 / 500 가까운 것넓고 높은 양수
🌟 WL (창높이) 변화 핵심
  • WL이 높아지면 → 영상 밝아짐 → 뼈 잘 보임
  • WL이 낮아지면 → 영상 어두워짐 → 공기 잘 보임
WW (창폭) 효과
  • WW 좁음 → 대조도 ↑
  • WW 넓음 → 대조도 ↓
  • WW 변화 ≠ 화질 변화 (디스플레이만 영향)

📐 계산 공식 암기법 (교수님 팁)

  • 둘 중 하나가 0일 때: 큰 값을 반 나눠서 양쪽에
    • 예: WW 100, WL 0 → -50 ~ +50
  • 둘 다 숫자일 때: WL에서 WW/2 빼고, 더하기

06관전압(kVp) vs 관전류(mAs)

  • 관전압(kVp) = 질 (에너지) → 대조도, 투과력 영향
  • 관전류(mAs) = 양 (광자 수) → 잡음, 피폭 영향

고관전압(kVp ↑) 효과

  • ✅ 투과력 ↑
  • ✅ 선속경화 ↓
  • ✅ 금속 인공물 ↓
  • ✅ 광자 결핍 ↓
  • ✅ 잡음 ↓
  • ❌ 대조도 ↓
  • ❌ 피폭 ↑↑ (제곱 비례)
  • ❌ 조영제 효과 ↓

고관전류(mAs ↑) 효과

  • ✅ 잡음 ↓
  • ✅ 대조도분해능 ↑
  • ❌ 피폭 ↑ (선형 비례)

저관전압(kVp ↓) 활용

  • 조영제 대조도 극대화 (CTA)
  • 소아 CT (피폭 감소)

08Pitch의 효과

⚠️ 핵심: Pitch는 분모! (매우 중요)
Pitch ↑ = 피폭 ↓ (함정 주의)

Pitch 증가(↑) 효과

  • ✅ 검사시간 단축
  • ✅ 피폭선량 감소
  • ✅ 움직임 인공물 감소
  • ❌ Z축 해상력 ↓
  • ❌ 잡음 ↑
  • ❌ 나선형/풍차 인공물 ↑
  • ❌ 단면감성윤곽 악화
🎯 기억법
  • Pitch ↑ = 빨리 지나감 → 시간·피폭 ↓, 해상력 ↓
  • Pitch ↓ = 천천히 지나감 → 시간·피폭 ↑, 해상력 ↑

11Bolus Tracking과 CTA

Bolus tracking 핵심

  • 목적: 동맥조영 효과 극대화
  • 방법: 대동맥 ROI에서 실시간 HU 측정
  • Threshold: 100~150 HU
  • 모니터링 스캔: 저선량
❌ 함정 표현 (모두 오답)
  • "200 HU threshold" ❌
  • "고선량 모니터링" ❌
  • "혈관외유출 예방" ❌ (무관)
  • "조영제 주입량 측정" ❌

복부 혈관 구분

  • 대동맥: 척추 왼쪽 전방, 원형 (영상에서는 오른쪽)
  • IVC (하대정맥): 반대편, 타원형
  • 환자 왼쪽 = 영상 오른쪽 (반전)

부위 구분 (복부 vs 흉부)

  • 오름대동맥 → 흉부
  • 복부대동맥 → 복부 축상면
3D CTA 품질 = 절편두께 ⭐

13피폭선량

피폭 증가 요인

  • 관전류 증가 ⭐ (선형, 가장 직접적)
  • 관전압 증가 (제곱)
  • Pitch 감소
  • Scan length 증가

피폭 감소 요인

  • 관전류 감소
  • 관전압 감소
  • Pitch 증가
  • AEC 사용
  • IR/DLR 재구성

AEC (Automatic Exposure Control)

  • 체형에 따라 mA 자동 조절
  • 피폭 감소 기술
  • 10~40% 피폭 감소
❌ 함정 정리 (모두 오답)
  • "높은 pitch = 피폭 증가" ❌ (pitch는 분모, 감소가 맞음)
  • "낮은 관전압 = 피폭 증가" ❌
  • "짧은 검사시간 = 피폭 증가" ❌
  • "AEC 사용 = 피폭 증가" ❌
"피폭 증가의 직접적 요인" = 높은 관전류 ⭐

14자주 나오는 함정 (오답 정리)

❌ 매트릭스 관련
  • "Matrix 증가 = 잡음 감소" ❌
  • 실제: 공간분해능 ↑, 잡음도 ↑
❌ 연부조직 알고리즘 관련
  • "Soft = 공간분해능 증가" ❌
  • 실제: 공간분해능 ↓, 대조도분해능 ↑
❌ Pitch 관련
  • "Pitch 증가 = 피폭 증가" ❌
  • 실제: Pitch 증가 = 피폭 감소 (분모!)
❌ 빔 경화 보정 장치
  • "콜리메이터 = 빔 경화 보정" ❌
  • 실제: 나비형 필터 (Bow-tie) ⭐
  • 콜리메이터는 빔 모양 제한
  • 필터는 빔 에너지 조절
❌ Window 관련
  • "창폭/창높이 차이 = 화질 차이" ❌
  • 실제: 디스플레이만 영향, 화질 자체는 동일
  • 단, 밝기 비교로 WL 높낮이는 판단 가능
❌ 관전압 관련
  • "고 kVp = 대조도 증가" ❌
  • 실제: 대조도 감소, 투과력 증가
❌ 3D 영상 관련
  • "3D 화질 차이 = 관전압/관전류" ❌
  • 실제: 절편두께가 주원인
❌ X선관 관련
  • "초점이 커야 한다" ❌
  • 실제: 열용적 용량이 1순위
❌ Ring artifact 관련
  • "Ring = 움직임" ❌
  • 실제: 검출기 정렬·보정 오류
❌ 28번 유형 (영상 밝기 판단)
  • "밝다 = 뼈 window" ❌
  • 폐 window도 밝게 보임 (WL이 상대적으로 높음)
  • 뼈 window는 WL이 훨씬 더 높음

영상 문제 접근법

  1. 영상 유형(2D axial/MPR/3D) 먼저 구분
  2. 시각적 특징(밝기·대조도·입체감) 파악
  3. 해당 유형의 화질 결정 요인 매칭
  4. 교수님 답 기준 우선

알고리즘 문제 접근법

  • 보기 구성에 따라 답 달라짐
  • "일반 조직" → Standard
  • "연부조직 특화" → Soft
  • "폐" → Lung/High
  • "뼈" → Bone
  • "관자뼈" → Ultra-high/Edge

매개변수 문제 접근법

  • 트레이드오프 관계 파악
  • 공간분해능 ↑ → 잡음 ↑
  • 대조도분해능 ↑ → 공간분해능 ↓
  • 하나 얻으면 하나 잃음
🌟 절편두께 문제 공식
  • 대부분 "얇게"가 답
  • 단, 잡음 감소만 "두껍게"
🌟 Pitch 문제 공식
  • Pitch는 분모
  • Pitch ↑ = 피폭 ↓, 시간 ↓, 해상력 ↓
🌟 인공물 문제 공식
  • 모양으로 판별
  • 별 모양 → 금속
  • 검은 줄 → 선속경화 → 필터로 해결
  • 동심원 → Ring → 검출기 교체
  • 풍차 → Windmill → Pitch ↓
  • 흐림 → Motion → 고정
